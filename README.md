# Titanic Veri Analizi Projesi

Bu proje, Veri Analizi dersi kapsamında hazırlanmıştır.  
Projede Kaggle platformundan alınan Titanic veri seti kullanılmıştır.

## Proje Amacı
Bu çalışmanın amacı, Titanic gemisindeki yolcuların hayatta kalma durumlarını
bazı özelliklere bakarak tahmin etmektir.

## Kullanılan Veri Seti
Veri seti Kaggle Titanic veri setidir.
Veri setinde yolcuların yaşı, cinsiyeti, yolcu sınıfı gibi bilgiler bulunmaktadır.

## Yapılan Çalışmalar
- Veri seti incelendi
- Eksik veriler dolduruldu
- Temel veri analizi yapıldı
- Makine öğrenmesi modeli kuruldu
- Model başarımı ölçüldü

## Kullanılan Yöntem
Bu projede sınıflandırma problemi çözülmüştür.
Model olarak Logistic Regression kullanılmıştır.

## Sonuç
Modelin doğruluk oranı hesaplanmış ve sonuçlar yorumlanmıştır.
Elde edilen sonuçlara göre bazı özelliklerin hayatta kalma üzerinde
etkili olduğu görülmüştür.
